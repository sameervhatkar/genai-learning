AI Life OS
=========

A long-term personal project to build an AI-powered life operating system —
starting from strong programming foundations and evolving into a
voice-controlled, decision-aware automation system.

This repository documents my **daily learning, experiments, and mini-projects**
as I work towards full AI-driven automation of digital and physical tasks.

--------------------------------------------------

VISION
------

The goal of this project is to build a system that can:

- Understand voice commands
- Reason before taking actions
- Automate everyday digital tasks (email, files, system commands, APIs)
- Control smart devices
- Learn from past behavior and preferences
- Act as a personal AI assistant, not just a chatbot

This is not a tutorial repository.
This is a **system-building journey**.

--------------------------------------------------

LEARNING STRATEGY
-----------------

- Consistent learning: **1 hour every day**
- Focus on fundamentals before advanced AI
- Build real, working components
- Document everything for weekend revision
- Improve clarity, not speed
